# 사업자대출 계산기 BY 정진에셋 이명진

아파트 시세와 기존 대출금을 기반으로 LTV 70%, 75%, 80% 한도를 자동으로 계산해주는 웹 애플리케이션입니다.

## 주요 기능

- **LTV 자동 계산**: 입력된 아파트 시세의 70%, 75%, 80% 대출 한도를 즉시 계산합니다.
- **점유 형태별 맞춤 계산**: 
  - **실거주**: 방공제(소액임차보증금) 금액을 차감한 실제 가용 한도 계산
  - **세입자 거주**: 세입자 보증금을 차감한 실제 가용 한도 계산
- **기존 대출 통합**: 기존 보유 중인 대출금을 합산하여 추가 대출 가능 금액을 산출합니다.
- **결과 리포트**: 계산된 LTV 구간별 한도와 상세 정보를 텍스트 형태로 요약하여 복사할 수 있습니다.
- **시각화**: 차트를 통해 기존 대출, 차감액, 추가 가능 금액을 직관적으로 비교할 수 있습니다.

## 기술 스택

- **Frontend**: React, TypeScript
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React

## 설치 및 실행

이 프로젝트는 React와 TypeScript로 작성되었습니다.

```bash
# 의존성 설치 (필요한 경우)
npm install

# 개발 서버 실행
npm start
```
